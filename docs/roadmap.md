# Roadmap

## Sprint 0: Foundation

- terminal assistant loop
- personality loading from YAML
- swappable LLM interface
- JSON memory store
- environment-based configuration
- tests for the core abstractions

## Sprint 1: Better Conversation State

- summarize long conversations
- improve memory retrieval
- add structured assistant events
- persist conversation transcripts

## Sprint 2: Voice Layer

- microphone input
- speech-to-text adapter
- text-to-speech output
- wake and sleep phrase detection

## Sprint 3: Platform Integrations

- Raspberry Pi deployment profile
- Bluetooth accessory control
- Spotify control
- GPS context ingestion

## Sprint 4: Motorcycle OS Features

- ride logging
- diagnostics integration
- contextual assistant actions
- safety and maintenance workflows
